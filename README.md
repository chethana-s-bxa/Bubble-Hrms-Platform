**HR Management System (HRMS)**

A complete HR Management System built to streamline employee management, document handling, and time tracking within an organization.

This system is divided into three core modules:

👤 Employee Management

📂 Document Management

⏱ Time Management

**System Overview**

The HRMS enables organizations to:

    Manage employee profiles and organizational hierarchy
    
    Handle employee documents securely
    
    Track attendance and time-related activities
    
    Maintain structured and validated employee data

1️⃣ **Employee Management Module**

This module handles complete employee lifecycle management.

🔹 Features

    Employee Profile Creation & Update
    
    Dynamic Organization Chart (Manager → Employee hierarchy)
    
    Personal Information Management
    
    Address Management (Current & Permanent)
    
    Emergency Contact Management
    
    Education & Work Experience Tracking
    
    Skills Management (Primary & Secondary)
    
    Band / Level Tracking
    
    Role-based Access (Admin / HR / Employee)
    
    Real-time form validations

🔹 Advanced Validations

    Indian Mobile Number validation
    
    IFSC code validation with auto bank detection
    
    Account number validation
    
    Duplicate experience prevention
    
    Date validation (No future dates)
    
    Postal code validation (multi-country support)

2️⃣ **Document Management Module**

Secure handling of employee documents.

🔹 Features

    Upload employee documents
    
    Categorize documents (ID Proof, Certificates, etc.)
    
    Download functionality
    
    View document metadata
    
    Secure access control
    
    API-based file storage

🔹 Supported Functionalities

    Fetch document list
    
    Download via secure endpoint
    
    Role-based access control
    
    Document type tagging

3️⃣ **Time Management Module**

Tracks employee working hours and attendance.

🔹 Features

    Attendance tracking
    
    Check-in / Check-out functionality
    
    Work duration calculation
    
    Leave tracking (optional extension)
